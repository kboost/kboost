## 🧭 Quién soy

Ingeniero Civil de Minas **(mención Ciberseguridad)** con experiencia en operaciones subterráneas en Chuquicamata, certificación Cisco, y un giro deliberado hacia el desarrollo de producto.

> Estoy combinando tres mundos que casi nadie une en Chile: **minería real**, **ciberseguridad** y **IA aplicada**, para construir software que las contratistas mineras necesitan pero hoy no tienen.

- 🏗️ **De servicio a producto** — vendo SaaS verticales en lugar de horas
- 🇨🇱 **Foco Chile primero**, expansión LatAm (Perú, Brasil) en roadmap
- 🚀 Construyo demos funcionales completas — el cliente toca el producto antes de firmar
- 📐 Stack moderno: Next.js 16, TypeScript, Tailwind v4, R3F para 3D técnico, Bun runtime
